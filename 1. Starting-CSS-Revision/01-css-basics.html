Concept 1: CSS Basics

What is CSS?

CSS stands for Cascading Style Sheets.

While HTML is used to structure a web document, CSS is used to dictate its style, layout, and presentation (colors, fonts, spacing, etc.).

CSS Syntax

A CSS rule consists of a selector and a declaration block.

h1 {
  color: blue;
  font-size: 24px;
}
  • Selector (h1): Points to the HTML element you want to style.
  • Declaration (color: blue): Contains a property and a value separated by a colon.

How to Add CSS to HTML

  1. Inline CSS: Added directly to an HTML element using the style attribute.
    This text uses inline CSS (color: green).

  2. Internal CSS: Defined within a <style> element, inside the <head> section of an HTML page. (This current page uses internal CSS mostly).

  3. External CSS: Defined in an external .css file, and linked inside the <head> using the <link> tag. This is the best practice for real projects!
