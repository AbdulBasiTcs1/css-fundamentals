Concept 3: The CSS Box Model

What is the Box Model?

All HTML elements can be considered as boxes. In CSS, the term "box model" is used when talking about design and layout.

The CSS box model is essentially a box that wraps around every HTML element. It consists of: Content, Padding, Borders, and Margins.

1. Content

The content of the box, where text and images appear. In this example, width is 250px and height is 50px.

CONTENT (250x50)

2. Content + Padding

Padding clears an area around the content. The padding is transparent. It sits inside the border.

Notice how the box got larger? Standard box-sizing adds padding & border to the base width!
Total width is now 250 + 20(left) + 20(right) = 290px!

CONTENT + PADDING

3. Content + Padding + Border

Border goes around the padding and content. We added a 10px solid red border.

Total width = 250(content) + 40(padding) + 20(borders) = 310px!

CONTENT + PADDING + BORDER

4. Complete: Content + Padding + Border + Margin

Margin clears an area outside the border. The margin is transparent. It separates this box from other elements.

We added a 30px top and bottom margin to push it away from the text. Note: Margin does not affect the size of the box itself, but the space around it.
