Concept 2: CSS Selectors

What are Selectors?

CSS selectors are used to "find" (or select) the HTML elements you want to style. They are the core of how CSS works.

1. Element (Tag) Selector

This whole paragraph is styled using the Element Selector (p { ... }). It targets all <p> tags on the page automatically.

2. Class Selector

You can use a class to target specific elements. This text has the class "highlight" which gives it a yellow background. Multiple elements across your page can have the same class, making it the most commonly used selector.

3. ID Selector

The main title of this page ("Concept 2: CSS Selectors") uses an ID Selector (#main-title { ... }). IDs must be unique; you should only use a specific ID once per page.

4. Grouping Selector

Notice how the <h2> and <h3> tags in this document share the same dark blue color? This is done using a Grouping Selector (h2, h3 { ... }) to write less code and keep things DRY (Don't Repeat Yourself).

5. Universal Selector

The universal selector (*) applies styles to every single element on the page. It's often used for resetting margins, padding, or setting global box models.
